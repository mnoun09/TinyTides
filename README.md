Tiny Tides is meant to be a calming game where you decorate your own aquarium tank. There's something so relaxing about aquariums and decorating in general, so that's how Tiny Tides came to be. 

This game was made with Godot. I'm still learning to navigate the software, and I especially had a hard time with creating movable objects. I heavily utilized global groups in this project. Tiny Tides was submitted to Milkyway Hack Club and is incomplete. There are so many features and additional decorations that I'd love to add in the future. One of the features I was working on was the ability to draw your own fish to add. Stay tuned for that!
